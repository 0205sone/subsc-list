# SubscList（サブスクリスト）

動画配信サービスや、アイドルのファンクラブなどの「定額課金（サブスクリプション）」を一元管理し、毎月の固定費を可視化するWebアプリケーションです。

## 開発の背景・経緯
私自身、複数のファンクラブや配信サービスに入会しており、「結局、月々いくら払っているのか？」が不透明になっていました。
特にファンクラブのような「年払い」のサービスを、月額換算して実質的な負担額を把握したいと考え、このアプリを開発しました。

## 公開URL
https://subsc-list-yg7n.vercel.app/

## 特徴と機能の説明

1.  **年払いの自動月割り計算**
    * ファンクラブなどの「年払い（例: 4,000円/年）」を登録すると、自動で「月額 333円」のように計算して表示します。これにより、実質的な月々の出費がひと目でわかります。

2.  **毎月の固定費合計の可視化**
    * 登録したすべてのサブスクの金額を合算し、トップページに大きく表示します。

3.  **セキュアなユーザー認証機能**
    * Supabase Authenticationを使用し、メールアドレスとパスワードによるログイン機能を実装しました。ユーザーごとに自分だけのデータを保存・管理できます。

## 工夫した点・苦労した点

### 工夫した点
* **UI/UX**: 毎日見ても疲れないよう、シンプルで清潔感のある青色を基調としたデザインにしました。
* **タグ表示**: 「年払い」か「月払い」かが直感的にわかるよう、色分けしたタグを表示させました。
* **自動化**: GitHub Actionsを設定し、Supabaseが停止しないように定期的なPing送信を自動化しました。

### 苦労した点
* **Tailwind CSSのバージョン対応**: 開発途中でTailwind CSSのバージョンによる設定エラーが発生しましたが、公式ドキュメントやエラーログを読み解き、適切なプラグイン（@tailwindcss/postcss）を導入することで解決しました。
* **デプロイ設定**: Vercelへのデプロイ時に環境変数の設定が必要である点を理解するのに時間がかかりましたが、正しくキーを設定することで動作させることができました。

## 既知の課題と今後の展望

* **編集機能の実装**: 現在は「追加」と「削除」のみなので、金額が変わった際に編集できる機能を追加したいです。
* **カテゴリ分け**: 「エンタメ」「学習」「生活」などでフィルタリングできるようにしたいです。
